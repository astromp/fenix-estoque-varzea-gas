# Canais de Venda — Várzea Gás

Este documento registra como os canais/personagens de venda devem ser tratados no Projeto Fênix Estoque.

## 1. Conceito

Os nomes utilizados na operação devem ser tratados como canais/personagens de venda quando representarem origem/responsável pela venda.

Na Várzea Gás, foram registrados como canais/personagens de venda:

- André;
- João;
- Rogério;
- Portaria.

## 2. Regra importante sobre Portaria

Portaria é canal de venda.

Não tratar Portaria como:

- retirada;
- conferência;
- portão físico;
- movimento intermediário;
- setor separado de estoque.

Portaria deve entrar no mesmo grupo dos demais canais/personagens de venda.

## 3. Uso no sistema

Cada venda pode ser associada a um canal/personagem para facilitar:

- conferência por responsável/canal;
- identificação de divergências;
- análise de movimentação por origem da venda;
- revisão de lançamentos no fechamento.

## 4. Diagnóstico por canal

Quando houver estoque inconsistente, o sistema deve conseguir ajudar o conferente a revisar as movimentações por canal/personagem.

Exemplo de pergunta útil do sistema:

```text
A divergência está concentrada em algum canal de venda, como André, João, Rogério ou Portaria?
```

## 5. Regra de interpretação

Sempre que o termo Portaria aparecer dentro do contexto da Várzea Gás, interpretar como canal de venda, salvo ordem contrária expressa.
