COMO TESTAR NO COMPUTADOR

1. Abra a pasta do projeto.
2. Abra o arquivo js/config.js.
3. Substitua os placeholders:
   - COLE_AQUI_A_URL_DO_SUPABASE
   - COLE_AQUI_A_ANON_PUBLIC_KEY
4. Salve o arquivo.
5. Abra index.html no navegador.
6. Clique em "Consultar status".

ATENÇÃO DE SEGURANÇA

Use apenas a chave anon public do Supabase.
Nunca coloque no frontend:

- service_role
- senha do banco
- DATABASE_URL
- connection string

REGRA DE OURO

Estoque fechado, turno encerrado.
Estoque inconsistente, revisar até corrigir.

STATUS OPERACIONAIS

sem abertura -> aberto -> fechado
aberto -> inconsistente -> corrigido -> fechado
